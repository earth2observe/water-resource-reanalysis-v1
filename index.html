eartH2Observe Water Resource Reanalysis v1

The eartH2Observe first water resources reanalysis (WRR-tier1) multi model ensemble was produced by 8 project partners and 2 external partners. The modelling systems include four land surface models, 5 global hydrological models and a simple water balance model and covers the period 1979 -- 2012. This first version of the WRR is based on the current modelling systems of each group forced with a state-of-the-art atmospheric reanalysis. This release is meant to set a benchmark dataset for future testing. A description of the multi model ensemble can be found in earth2observe deliverable 5.1 and Schellekens et. al. (under discussion).

Model Verification Results

The multi model ensemble was verified against benchmark data for evapotranspiration, terrestrial water storage anomaly (TWSA), soil moisture index, snow water equivalent (SWE) and snow cover (SC). The verification was done with the help of the tools from The International Land Model Benchmarking Project (ILAMB, Luo et al., 2012)).

Model Data Access

Data Access Methods

All of the WRR1 model data is available for download by a number of methods:

  • WCI Portal View - this tool offers a method to explore the data visually; you can also analyse, extract and download data for specific regions using this tool.
  • Individual & Aggregated Files - access to the daily and monthly files using THREDDS via these protocols:
    • Web Map Service (WMS)
    • Web Coverage Service (WCS)
    • Opendap
    • HTTP Download
